## Update dotfiles

1. clone this repo then

   ```bash
   make dot
   ```

   or

   ```bash
   ./index.mjs
   ```

## Install Script For Linux

> run with 'npx', 'pnpm dlx'

```bash
pnpm dlx zx https://cdn.jsdelivr.net/gh/jungai/dotfiles@latest/install_linux.mjs
```
