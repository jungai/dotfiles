## Update dotfiles

1. clone this repo then

   ```bash
   make dot
   ```

   or

   ```bash
   ./index.mjs
   ```

## Install Script For Linux

> required `zx`

```bash
zx https://cdn.jsdelivr.net/gh/jungai/dotfiles@latest/install_linux.mjs
```
